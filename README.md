# Runway

Inteligencia de gastos y suscripciones para indie founders / freelancers.
Importás tus transacciones (CSV), la IA las categoriza por significado,
detecta suscripciones zombie/duplicadas, alerta de subidas de precio y
proyecta tu runway (meses de pista restantes).

Stack: Next.js (App Router) · Tailwind · Supabase (Postgres + RLS + pgvector + Edge Functions) · Vercel.

Estado: MVP en diseño.
